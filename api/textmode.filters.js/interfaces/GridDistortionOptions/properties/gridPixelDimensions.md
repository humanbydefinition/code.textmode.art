# Property: gridPixelDimensions

```ts
gridPixelDimensions: [number, number];
```

Grid size in pixels as `[width, height]`.

Typically calculated as:
`[t.grid.cols * t.grid.cellWidth, t.grid.rows * t.grid.cellHeight]`

## Default

```ts
[640.0, 320.0]
```
