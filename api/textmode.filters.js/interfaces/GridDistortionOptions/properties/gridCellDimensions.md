# Property: gridCellDimensions

```ts
gridCellDimensions: [number, number];
```

Grid dimensions as `[columns, rows]`.

Should match your textmode grid dimensions.

Maximum value: `[128, 128]`

## Default

```ts
[80.0, 40.0]
```
