# Property: gridOffsetDimensions

```ts
gridOffsetDimensions: [number, number];
```

Grid offset in pixels as `[offsetX, offsetY]`.

Use `[t.grid.offsetX, t.grid.offsetY]` to match your grid position.

## Default

```ts
[0.0, 0.0]
```
