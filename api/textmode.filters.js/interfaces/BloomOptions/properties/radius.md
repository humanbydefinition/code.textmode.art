# Property: radius

```ts
radius: number;
```

Size of the glow spread in pixels.

Larger values create a wider, softer glow.

- `1.0` = tight glow
- `4.0` = moderate spread (default)
- `10.0+` = wide, diffuse glow

## Default

```ts
4.0
```
