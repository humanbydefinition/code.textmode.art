# Property: lineWidth

```ts
lineWidth: number;
```

Width of the lines relative to the spacing.

- `0.0` = very thin lines (mostly transparent)
- `0.5` = equal line and gap width
- `1.0` = solid (no gaps)

## Default

```ts
0.5
```
