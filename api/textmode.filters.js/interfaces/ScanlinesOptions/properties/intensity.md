# Property: intensity

```ts
intensity: number;
```

Opacity/darkness of the scanlines.

- `0.0` = invisible lines
- `0.75` = clearly visible (default)
- `1.0` = maximum darkness (solid black lines)

## Default

```ts
0.75
```
