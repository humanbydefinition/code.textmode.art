# Property: amount

```ts
amount: number;
```

Contrast multiplier.

- `1.0` = normal contrast (no change)
- `> 1.0` = more contrast (darker darks, brighter brights)
- `< 1.0` = less contrast (more gray/washed out)
- `0.0` = solid gray

## Default

```ts
1.0
```
