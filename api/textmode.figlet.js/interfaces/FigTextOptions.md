# Interface: FigTextOptions

Layout options for rendering FIGlet text.

## Properties

| Property | Description |
| ------ | ------ |
| [horizontalLayout](FigTextOptions/properties/horizontalLayout.md) | Override the font's default horizontal layout. |
| [verticalLayout](FigTextOptions/properties/verticalLayout.md) | Override the vertical composition mode for multi-line FIGlet layout. |
| [direction](FigTextOptions/properties/direction.md) | Override the effective print direction for layout. |
| [wrap](FigTextOptions/properties/wrap.md) | Optional wrap strategy for multi-line layout. |
| [maxCols](FigTextOptions/properties/maxCols.md) | Maximum rendered columns per logical line before wrapping occurs. |
| [charColor](FigTextOptions/properties/charColor.md) | Override or resolve the foreground color for each rendered FIGlet cell. |
| [cellColor](FigTextOptions/properties/cellColor.md) | Override or resolve the background color for each rendered FIGlet cell. |
