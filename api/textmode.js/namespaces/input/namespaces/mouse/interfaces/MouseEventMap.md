# Interface: MouseEventMap

Event map for all mouse events emitted by the MouseInput.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="doubleclicked"></a> `doubleClicked` | [`MouseEventHandler`](../type-aliases/MouseEventHandler.md) | Fires when the mouse is double-clicked. |
| <a id="mouseclicked"></a> `mouseClicked` | [`MouseEventHandler`](../type-aliases/MouseEventHandler.md) | Fires when the mouse button is clicked (full press + release). |
| <a id="mousedragged"></a> `mouseDragged` | [`MouseEventHandler`](../type-aliases/MouseEventHandler.md) | Fires when the mouse moves while a button is held down. |
| <a id="mousemoved"></a> `mouseMoved` | [`MouseEventHandler`](../type-aliases/MouseEventHandler.md) | Fires when the mouse moves over the canvas. |
| <a id="mousepressed"></a> `mousePressed` | [`MouseEventHandler`](../type-aliases/MouseEventHandler.md) | Fires when a mouse button is pressed down. |
| <a id="mousereleased"></a> `mouseReleased` | [`MouseEventHandler`](../type-aliases/MouseEventHandler.md) | Fires when a mouse button is released. |
| <a id="mousescrolled"></a> `mouseScrolled` | [`MouseEventHandler`](../type-aliases/MouseEventHandler.md) | Fires when the mouse wheel is scrolled. |
