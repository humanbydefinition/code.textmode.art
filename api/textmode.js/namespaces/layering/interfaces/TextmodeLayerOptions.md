# Interface: TextmodeLayerOptions

Options for layers created with [TextmodeLayerManager.add](../classes/TextmodeLayerManager/methods/add.md).

## Properties

| Property | Description |
| ------ | ------ |
| [blendMode](TextmodeLayerOptions/properties/blendMode.md) | Blend mode used when compositing this layer. Defaults to `'normal'`. |
| [fontSize](TextmodeLayerOptions/properties/fontSize.md) | Font size for the layer's glyph source. Defaults to `16`. |
| [fontSource](TextmodeLayerOptions/properties/fontSource.md) | Source for the font to use in this layer. |
| [offsetX](TextmodeLayerOptions/properties/offsetX.md) | Horizontal layer offset in pixels. Defaults to `0`. |
| [offsetY](TextmodeLayerOptions/properties/offsetY.md) | Vertical layer offset in pixels. Defaults to `0`. |
| [opacity](TextmodeLayerOptions/properties/opacity.md) | Layer opacity from `0` (transparent) to `1` (opaque). Defaults to `1`. |
| [rotationZ](TextmodeLayerOptions/properties/rotationZ.md) | Z rotation in degrees around the layer center. Defaults to `0`. |
| [visible](TextmodeLayerOptions/properties/visible.md) | Whether the layer is visible. Defaults to `true`. |
